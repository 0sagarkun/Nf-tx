# 🎬 Netflix TV Login Bot — Ultimate Edition

A powerful Telegram bot that automates Netflix TV activation using cookie-based authentication.  
Built for speed, reliability, and ease of use — with a credit system, cookie vault, and full admin panel.

---

## ✨ Features

- **Instant TV Login** – Just send your 8-digit TV code and the bot does the rest.
- **Cookie Vault** – Store thousands of Netflix cookies in a SQLite database, auto-rotated per login.
- **Credit System** – Each user gets 3 free credits per 12 hours (configurable).
- **Real‑time Status** – Animated progress indicator while the bot works.
- **Rich Account Info** – Displays country, plan, payment details, member since, streaming quality, and more.
- **Admin Panel** – Upload cookies via `.txt`, `.json`, or `.zip`; manage users; broadcast messages; monitor stats.
- **Inline Keyboard** – Easy navigation for users.
- **Proxy Support** – Use proxies from `proxy.txt` to avoid IP bans.
- **Multi‑Account Parsing** – Upload a file with dozens of cookies – the bot extracts all valid ones.

---

## 📦 Requirements

- Python 3.7+
- Telegram Bot Token (from [@BotFather](https://t.me/BotFather))
- SQLite3 (built‑in)
